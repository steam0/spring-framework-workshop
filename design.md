# Presentation Design

Static website based on the Norsk Tipping Design System (NTDS).

## Layout

### Header
- Fixed top bar, white background
- Home icon (outline style) on the left
- "Norsk Tipping" logo text on the right, haiti colored, bold
- Subtle bottom border (cloud)

### Landing / Navigation
- Section heading: "Spring Framework Workshop" in haiti, bold
- Topic buttons displayed as large rectangular cards in a horizontal row (wrapping on small screens)
- Each button links to the corresponding topic section on the page
- Buttons use varied NTDS brand colors for visual distinction (haiti, blue, green, red, yellow, pink, etc.)
- Button text is bold, centered, white on dark backgrounds, dark on light backgrounds
- Border-radius: 12px

### Content Sections
- Each `#` heading from presentation.md becomes a section on the page
- Section titles: large, bold, haiti color, with generous top margin
- Content scrolls vertically (normal webpage, not slides)

## Typography

- **Font family:** "Inter", system-ui, -apple-system, sans-serif
- **Headings:** bold, haiti color
- **Body text:** coal, 1.6 line-height
- **Code font:** "JetBrains Mono", "Fira Code", monospace

## Components

### Blockquotes / Info Boxes
- Background: green-light (#E8F5E0)
- Border-radius: 12px
- Padding: 1.2rem 1.5rem
- Text: coal, normal weight
- No left border — full rounded box like NT's info panels

### Code Blocks
- Background: haiti (#2C1A45) — dark purple
- Border-radius: 12px
- Padding: 1.5rem 2rem
- Font: JetBrains Mono
- Syntax highlighting colors on haiti background:
  - **Keywords** (class, return, new, private, final): #FF5797 (pink)
  - **Annotations** (@Service, @Bean, etc.): #FFCF3F (yellow)
  - **Strings:** #00932F (green)
  - **Types/classes:** #ffffff (white, bold)
  - **Methods/functions:** #FFCF3F (yellow)
  - **Comments:** #A29EA8 (steel)
  - **Default text:** #E8E6EB (cloud)
  - **Numbers:** #FF7438 (orange)

### Tables
- Header row: haiti background, white text
- Body rows: alternating white / smoke
- Cell borders: 1px solid cloud
- Cell padding: 0.75rem 1.25rem
- Border-radius: 12px on outer table container

### Bullet Points
- Cloud-colored disc markers
- Left margin: 1.5rem
- Spacing between items: 0.6rem
- Line-height: 1.6

### Inline Code
- Background: smoke
- Color: haiti
- Padding: 0.15em 0.4em
- Border-radius: 4px
- Font: JetBrains Mono

### Code Language Toggle
- When consecutive code blocks use different languages (kotlin + java), group them into a tabbed container
- Kotlin is the **default** (shown on load)
- Tab bar sits on top of the code block, inside the haiti background
- Tabs: "Kotlin" and "Java" text buttons
- Active tab: white text, underlined with yellow
- Inactive tab: steel text, no underline
- Clicking a tab on *any* code block switches *all* code blocks on the page to that language (global toggle)
- Tab bar padding: 0.5rem 2rem, no extra background — flush with the code block
- Non-togglable code blocks (yaml, bash, or single-language blocks) render as before with no tabs
