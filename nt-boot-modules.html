Getting Started

To use nt-boot, set nt-boot-starter-parent as your Maven parent. This gives you Spring Boot, nt-boot-core, nt-boot-web, Kotlin support, and build tooling — all pre-configured.

<?xml version="1.0" encoding="UTF-8"?>
<project ...>

    <parent>
        <groupId>no.norsktipping.boot</groupId>
        <artifactId>nt-boot-starter-parent</artifactId>
        <version>4.4.0</version>
    </parent>

    <groupId>no.norsktipping</groupId>
    <artifactId>my-service</artifactId>
    <version>1.0.0</version>

    <properties>
        <java.version>21</java.version>
    </properties>

</project>

What the parent gives you

You don't need to declare these — they come automatically:

nt-boot-core
JSON logging, metrics, health probes, NtApplicationConfig
nt-boot-web
Tracing headers, request logging, CORS
spring-boot-maven-plugin
With build-info goal — required for NtApplicationConfig
Kotlin + JaCoCo + git-commit-id
Compiler, allopen plugin, and test coverage pre-configured
